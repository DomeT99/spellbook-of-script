# Spellbook of Script 📜

Spellbook of Script is a collection of bash scripts designed to automate everyday work. Whether you're looking to streamline your workflow or perform repetitive tasks more efficiently, you'll find useful scripts here.

## What's Inside? 🗂️

- **Automation Scripts**: Scripts to automate daily tasks.
- **Utility Scripts**: Handy scripts for various utilities.
- **System Scripts**: Scripts to manage and monitor system operations.

## How to Contribute 📖
If you want to add more scripts, follow these steps:

1. Fork the repository
2. Clone the repository to your local machine
3. Create a new branch

```
 git checkout -b <branch-name>
```

4. Make your changes
5. Commit and push your changes

```
 git add .
 git commit -m "commit message"
 git push origin <branch-name>
```

6. Create a pull request
7. Wait for the pull request to be reviewed and merged
